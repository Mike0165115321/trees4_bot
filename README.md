# 🌳 Trees4All Automation Bot

กรอกข้อมูลต้นไม้อัตโนมัติบน trees4allthailand.org

---

## ติดตั้งครั้งแรก (ทำครั้งเดียว)

เปิด Command Prompt หรือ PowerShell แล้วรันทีละบรรทัด:

```
pip install playwright
playwright install chromium
```

---

## วิธีรัน

```
python trees_bot.py
```

Bot จะถามข้อมูล:
1. เบอร์โทร / รหัสเกษตรกร
2. รหัสผ่าน
3. ชื่อผู้จดบันทึก (default: ชยพล พรมสะวะนา)
4. ชื่อผู้สำรวจ (default: กรรณิการ์ คำนาน)
5. สัดส่วนคะแนนสุขภาพ (%, ปรับได้)

Bot จะกรอก **ทุกต้นที่ยังไม่ได้บันทึก** จนหมดอัตโนมัติ (Process All Mode)

---

## สิ่งที่ Bot ทำ

- เปิด browser จริงให้ดูได้ตลอด
- Login อัตโนมัติ
- วนกรอกแต่ละต้น:
  - เลือกต้นบนสุดที่ยังไม่ได้กรอกจากลิส
  - ถ้าเจอชนิดต้นไม้อยู่แล้ว → ใช้เลย / ถ้าว่าง → เลือกอันแรก
  - สุขภาพต้นกล้า → สุ่มตามสัดส่วนที่ตั้งไว้
- หน่วงเวลาระหว่าง record แบบ random (ดูเป็นธรรมชาติ)
- แสดงสรุปผลเมื่อเสร็จ (จำนวนสำเร็จ / error / เวลาที่ใช้)

---

## ปรับค่าสุขภาพ (ถามตอนเริ่มรัน)

```
  คะแนน 3 (สุขภาพดี)   % : 80    (default)
  คะแนน 2 (ปานกลาง)    % : 15    (default)
  คะแนน 1 (แย่)         % :  5    (default)
```
