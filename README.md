# 🌳 Trees4All Automation Bot v6 (Ultra-Autonomous)

ระบบกรอกข้อมูลต้นไม้อัตโนมัติบน Trees4All Thailand แบบเน้นความเร็วและเสถียรภาพสูงสุด

---

## ✨ ความสามารถล่าสุด (v6)

- **🚀 Turbo Speed:** ปรับจูนความเร็วให้ทำงานได้ 2.5 - 3 ต้นต่อนาที (เลียนแบบความเร็วผู้เชี่ยวชาญ)
- **🌑 Headless Mode:** รองรับการทำงานเบื้องหลัง (ไม่ต้องเปิดหน้าต่าง Chrome กวนใจ)
- **⚙️ Config System:** ตั้งค่าผ่าน `config.txt` เพียงครั้งเดียว เริ่มรันได้ทันทีไม่ต้องกรอกใหม่
- **🛠️ Auto-Focus Fix:** แก้ปัญหาหน้าผู้สำรวจค้าง โดยการ Click & Blur อัตโนมัติ
- **🤖 Ultra-Autonomous:** เลือกต้นไม้ กรอกชนิด สุ่มสุขภาพ และวนลูปจนจบรายการเอง 100%

---

## 🚀 วิธีการใช้งาน

1. แก้ไขข้อมูลใน [config.txt](file:///c:/Users/Gigabyte/Downloads/files/trees_bot_v6/config.txt) (เบอร์โทร, รหัสผ่าน, ชื่อผู้บันทึก)
2. รันคำสั่ง:
   ```bash
   python trees_bot.py
   ```
3. บอทจะเริ่มทำงานอัตโนมัติจนกว่าจะหมดรายการในลิสต์

---

## ⚙️ การตั้งค่าใน config.txt

- **headless=true:** ทำงานเบื้องหลัง (แนะนำสำหรับระบบที่เสถียรแล้ว)
- **headless=false:** เปิดหน้าต่าง Browser ให้ดูการทำงาน (ค่าเริ่มต้น)
- **health_3, 2, 1:** ตั้งสัดส่วนการสุ่มคะแนนสุขภาพ (รวมกันต้องได้ 100)
