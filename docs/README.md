# Trees4All Command Center Documentation

ยินดีต้อนรับสู่คู่มือการใช้งานและการพัฒนาของโปรเจค **Trees4All Command Center** 

โฟลเดอร์ `docs/` นี้จะถูกใช้เพื่อเก็บเอกสารเจาะลึก (Deep Dive Documentation) สำหรับโครงสร้าง, API, Database Schema, และ Bot Logic ของโปรเจค เพื่อให้เป็นมาตรฐานและพร้อมส่งมอบหรือพัฒนาต่อในอนาคต

## โครงสร้างเอกสารที่วางแผนไว้ (Draft)
- `architecture.md` - ภาพรวมของระบบ (Frontend, Backend, Playwright)
- `database.md` - โครงสร้าง SQLite และ Foreign Keys
- `api_reference.md` - คู่มือ API ที่ใช้งานใน Dashboard
- `bot_logic.md` - สเต็ปการทำงานของ Playwright
- `setup_guide.md` - วิธีการติดตั้งและรันโปรเจคลงเครื่องใหม่
